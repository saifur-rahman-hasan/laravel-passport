## Installation Guide

* `cp .env.example .env`
* `composer update`
* `php artisan key:generate`
* change your database connection from `.env` according to your need
* `php artisan migrate`
* `php artisan migrate --path=database/migrations/passport`
* `php artisan passport:install`
* `yarn install`
* `yarn dev`
